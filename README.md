## Autopilot Modes — Quick Reference

| Say This | What Happens |
|---|---|
| `NEXT STEPS MODE` | Tells Bill the single most important thing to do right now |
| `CALL MODE` | Paste a transcript — Claude formats and saves it to calls/ |
| `SYNC MODE` | Claude searches Google Drive for new Landon recordings and pulls them |
| `PROPOSAL MODE` | Claude builds a meeting agenda and talking points for Landon |
| `BUILD MODE` | Claude breaks a roadmap item into specific tasks with acceptance criteria |
| `BASELINE MODE` | Claude documents current Bass Forecast metrics before work begins |
| `EMAIL MODE` | Claude drafts any email to Landon |
| `REVENUE MODE` | Claude calculates impact and Bill's earnings for any asset |
| `WEEKLY UPDATE MODE` | Claude drafts a progress report for Landon |

## How To Add A New Call

**Option 1 — Automatic via Google Drive:**
```
SYNC MODE
```
Claude searches Drive for new Landon Bass Forecast recordings and pulls them automatically.

**Option 2 — Manual paste:**
```
CALL MODE
```
Paste the transcript or summary. Claude formats it into the standard template and saves to calls/ with date prefix.

**After either option:**
Claude will extract key decisions, Landon's exact words, action items, open questions, and any new roadmap items. README calls index gets updated.
